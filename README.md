# 微博爬虫

爬取指定用户尽可能全面的信息。

## 已实现的功能

抓取用户个人基本信息。

抓取用户指定时间后的所有微博，包括被转发的微博，支持长微博正文、图片ID和文件。

抓取的微博和下载的附属图片保存到 output
